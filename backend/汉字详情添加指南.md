## 如何继续添加

### 方法一：编辑脚本批量添加

编辑 `batch_add_details.py` 文件，在 `character_details` 字典中添加更多汉字：

```python
character_details = {
    '新字': {
        'pinyin': '拼音',
        'definition': '释义1\n释义2\n释义3',
        'words': '词语1\n词语2\n词语3\n词语4\n词语5',
        'sentences': '例句1。\n例句2。\n例句3。'
    },
    # 添加更多...
}
```

然后运行：
```bash
cd backend
python3 batch_add_details.py
```

### 方法二：通过Web界面添加

1. 打开汉字库管理页面
2. 点击任意汉字
3. 在详情弹窗中点击"编辑"
4. 填写：
   - 拼音
   - 释义
   - 组词（每行一个）
   - 造句（每行一个）
5. 点击"保存"

### 方法三：使用API批量导入

创建JSON文件，然后通过API导入：

```python
import requests
import json

# 准备数据
data = {
    'character_id': 1,  # 汉字ID
    'pinyin': 'hàn',
    'definition': '释义内容',
    'words': '词语1\n词语2',
    'sentences': '句子1。\n句子2。'
}

# 调用API
response = requests.put('http://localhost:5000/api/characters/1', json=data)
print(response.json())
```

## 格式说明

### 拼音
- 格式：`pīn yīn`
- 多音字用 `/` 分隔：`de / dí / dì`

### 释义
- 每个释义占一行
- 用 `\n` 分隔
- 示例：
  ```
  1. 第一个意思。
  2. 第二个意思。
  3. 第三个意思。
  ```

### 组词
- 每个词语占一行
- 用 `\n` 分隔
- 建议5-10个常用词语
- 示例：
  ```
  汉字
  汉语
  汉族
  汉朝
  好汉
  ```

### 造句
- 每个句子占一行
- 用 `\n` 分隔
- 建议3-5个例句
- 示例：
  ```
  汉字是世界上最古老的文字之一。
  我们要学好汉语。
  汉族是中国人口最多的民族。
  ```

## 数据库字段

- `pinyin`: TEXT - 拼音
- `definition`: TEXT - 释义
- `words`: TEXT - 组词（换行分隔）
- `sentences`: TEXT - 造句（换行分隔）

## 提示

1. **优先级**：先为常用字和错题库中的字添加详情
2. **质量**：注重准确性和实用性

4. **来源**：可参考新华字典、现代汉语词典等权威工具书
